To insert a link, I add an "A" tag and then a href followed by an equal sign. I add a destination file inside the quotation marks for the link to route. I then add a text to refer to the user what they are clicking on. The text in our lab contains "Return Home". Finally, I add the closing tags to complete the route. CNN Website

For images, I start off with an img. Then I add the source of the image with "src" followed by an equal sign. The source could be a local image or a URL. Then I add an alternative text in case the image is not available. It could contain any text related to the image or project. The source has to be followed by the equal sign as well and the text must be included in a quotations. Additionally, I adjusted the width of the image because it was too large for the webpage. The width is handled just like the source, width followed by an equal sign and size within quotation marks in pixels. Example below..
